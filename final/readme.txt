first of all Camera and Seria communication of raspberry pi is required to enable 

This is a manual process

run the following command in terminal and enabe camera and serial using the steps

sudo raspi-config
 
1. Go to  Interfacing Options
2. Select Camera >>Yes >>Ok
3. Again go to  Interfacing Options
4. Select Serial >>( Would you like a login shell to be accessible over serial?) No >>(Would you like the serial port hardware to be enabled?)Yes
